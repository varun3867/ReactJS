MongoDB:
-----------
	- It is "NoSQL" Database
	- It is supports the JSON
	- It is light weight Database
	- By default MonogoDB database running on port 27017
	- MongoDB database supports the "mongodb" protocol
	- MongoDB provides "Cloud Environment" in the form of "Mongodb Altas".

SQL vs NoSQL :
--------------
	SQL									MySQL
	-------------------------------------------
Oracle, MySQL, SQLServer			MongoDB, CassandraDB, RabbitMQ
Databases							Databases
Tables								Collections
primary Key							_id
records								Documents


Front-end cannot talk with the database directly.
we have to use backend technologies like java, NodeJS, PHP etc.,

NodeJS:
--------
	- It is backend technology
	- By using NodeJS, we can develop servers
	- we will develop NodeJS applications with the help of JavaScript
	- NodeJS, supports the modules/libraries
	- we will download modules/libraries with the help of yarn tool
	
Modules:
---------
	1)express
		- used to develop the REST apis.
		Ex: GET, POST, PUT, DELETE
	2)mongodb
		- used to connect to mongodb database
	3)cors
		- used to enable the communication between different ports.
	4)jwt-simple
		- used to generate the tokens
		- These tokens are used to maintain the token based Authentication system
		
		
	> yarn add express mongodb cors jwt-simple
